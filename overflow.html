overflow has visible, hidden, auto, scroll. It works for block level element, with specified height.

visible is the default value, which will display the content outside the box.
hidden will hide the outside box content.
auto would be the ideal thing here, which will create a sidebar if needed.
scroll will always make the sidebar, even the outside content does not exist.

Also, to have scroll for left and right, we need overflow-x and will have the same properties as overflow itself.
