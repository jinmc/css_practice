Emmets in VSCode

VS code emmets are helpers that can increase your productivity, by having shortcusts. Pressing ! + tab will create a whole html document fresh.

Also, div + tab will make a div element, just as any other elements. div*10 + tab will make ten elements, div>h1 + tab will make h1 element inside div tag.

h1.heading + tab will make a h1 element with a class of heading, and similarily, h1#heading + tab will make a h1 element wiht a id of heading.
