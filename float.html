Float specifies how an element should float. Float items jump out of normal flow. Absolutely positioned elements ignore the float property! Also float needs width and height, as it's a block property.

We see that the border top and bottom are right together, because floats get out of normal flow. There are three ways to fix it.

The first way is to put overflow: auto in container.
The second way is to have a clear element and have attribute clear:both;
The third way is to have a class in the parent container and use that class to use the pseudo element after and make content, block element, and clear both attribute.
